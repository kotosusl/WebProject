Перед пользованием приложением убедитесь, что у вас установлен Python 3 и все библиотеки, указанные в requirements.txt.
Для установки библиотек напишите в терминале pip install -r requirements.txt

Для запуска необходимо в каталоге проекта создать файл TOKEN.py, в котором должна находиться переменная TOKEN типа str, где нужно поместить токен бота в телеграме.
Для корректной работы должна быть папка db и в файлах .py, subject_to_code.json и папке data не должно быть изменений.
!Несоблюдение требований выше не гарантирует работу программы!

Для пересоздания базы данных достаточно удалить её из папки db и перезапустить программу.

Чтобы бот начал работу, запустите файл telegram_bot.py.
